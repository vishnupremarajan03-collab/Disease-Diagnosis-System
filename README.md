Disease Diagnosis System:-
A machine learning based disease prediction system designed to assist users in identifying potential diseases based on symptoms and health indicators. The project focuses on applying data-driven techniques to build an intelligent diagnostic support tool.

Project Overview:-
The Disease Diagnosis System leverages Random Forest algorithm to analyze input symptoms(132 symptoms) and predict possible diseases(42 diseases). The goal of this project is to explore how AI/ML techniques can support early health assessment and decision making.
This project was developed as part of my academic and practical exploration of:
1. Machine Learning
2. Data Processing & Feature Engineering
3. Predictive Modeling

Key Features:-
✔ Symptom-based disease prediction
✔ Data preprocessing & cleaning pipeline
✔ Machine learning model implementation
✔ Extendable architecture for future improvements
